## 工夫した点

- 工夫した点を書いてください（必須）．


ユーザーが自分の描いた絵でアイコンを作成できるようにしたこと。さらに返信機能をつけることでユーザー同士の

繋がりを促進した。手書きのアイコンには愛着と個性が出るので、人それぞれユニークなアイコンになって面白いと

思った。
